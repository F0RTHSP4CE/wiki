# Функционал 

На текущий момент LED-панель визуализирует [Game of Life](https://conwaylife.com/wiki/Conway%27s_Game_of_Life) и выводит информацию об имени резидента, открывшего входную дверь. 

К LED-панели подключен микроконтроллер ESP32-C3, на котором реализован WEB-сервер для принятия и отображения содержимого HTTP-POST запроса - [пример](https://t.me/f0rthsp4ce_l1ve/238). То есть существует возможность отобразить на экране любой текст, если сформировать корректный HTTP-POST запрос. Пример запроса:

`curl -X POST 'http://10.0.24.33/text?x=1&y=1&timeout=10000&text=hello+world`

Необходимо находиться в резидентской сети для доступа к интерфейсу LED-панели.

Исходный код проекта доступен в [репозитории F0rthSp4ce](https://github.com/f0rthsp4ce/ledka).
