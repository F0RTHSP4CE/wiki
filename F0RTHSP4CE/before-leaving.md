# Before leaving

**Check that power is off (turn off):**
- Soldering table power strip
- Heating
- Air conditioner
- Main lights

**Leave powered (turn on, if not):**
- Ventilation fan in basement window (helps with air circulation)
- Aquarium (filter, heater and timer for lights)
- Media gallery and Gigatron - screens facing outside in the window by the aquarium
- No need to check, but we usually leave on auxiliary LED lighting, which includes:
  - LED matrix running Conway's Game of Life
  - WLED strip in basement
  - Bottle display in basement
  - LED strip on shelves in openspace
  - Tasmota smart bulb in corridor
  - Lamp next to 3D printer

**Check that closed:**
- door in corridor
- window in openspace/kitchen
- basement door
- entrance door

**Extras:**
- **throw away trash**, if bins are full (new bags can be found in top drawer in kitchen), places to dispose trash can be found [here](https://t.me/c/1900643629/7346/7366)
- clean up, especially tables in openspace
- do not leave dirty dishes and cups
